Auto_AVI is an automation python script that moves avi files from local folder to the remote folder.

To use the script Python will need to be installed.

create a folder on the remote desktop
open the command line and type "python auto_avi.py [folder name]" and press enter       *(If your folder name contains spaces or special characters add quotation marks around your folder name. Do not add brackets.)
the program will run and copy files from your local folder to the remote folder that was entered in the command.

You can keep the command prompt open and press up on the arrow key to bring up the program command and use again to update the remote folder.
